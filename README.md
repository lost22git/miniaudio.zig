my learning demonstration of Zig code interacting with C code

[API DOC](https://lost22git.github.io/miniaudio.zig)

## Usage

### Build and Run app

```sh
zig build run -Doptimize=ReleaseFast -- your.mp3
```

### Build Docs

```sh
zig build docs
```
